## Pages

* Home: 

    ** Add new todo list
        
        - Input fields to insert new todo item
        - Deadline fields
        - Button to submit

    ** List all todo item include: 

        - Not yet 
        - Processing 
        - Done

* Not Yet:
    ** List item not yet

* Process:
    ** List item in processing

* Done:
    ** List item has done
